# MarkMe

Lightweight markdown editor.

# Features

- File explorer
- iCloud Drive support
- Edit & Preview markdown files
- Preview as MindMap
- Themes, fonts and text sizes
- Import from Files app
- Export to HTML/PDF/Image
- Handy keyboard toolbar for editing
- Custom style sheet
