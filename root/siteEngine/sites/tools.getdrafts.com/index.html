Drafts Tools

The Drafts Tools site is a collection on online resources to assist in the creation of Drafts actions and themes.

Theme Builder

The Theme Builder is a web-based tool to make modifications to Drafts Themes. It is most useful for loading existing themes (either from an exported file, or directly from the Drafts Directory), making making minor tweaks to suit your tastes, and importing the results into Drafts.

Action Wizards

Action wizards are simplified, web-based forms to help create custom Drafts actions. These actions could be created directly in Drafts, but wizards make it simpler to get started with custom actions.

Available Wizards

  • Messages Action Wizard
    • Create iMessage actions configured to specific recipients, so you are one tap away from sending to friends, family, or groups you message often.
  • Mail Action Wizard
    • Create email actions with pre-defined recipients. Great for group emails, with support for To, CC, and BCC recipients.
