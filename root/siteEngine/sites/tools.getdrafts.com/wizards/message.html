Message Action Wizard

Simplified form to create and install a Drafts action to send an iMessage to pre-defined recipient list. To use, complete the form below and click “Install Action” button on a device with Drafts installed.

Basic Configuration
Message Recipients
Provide a comma-separated list of phone numbers and/or iMessage enabled email addresses to send the message.
Message Content
Template to define content of message. Use [[draft]] for full text of current draft, [[title]] for first line. Learn more about Drafts template tags.
Install

After configuring options above, use the "Install Action" button to install as an action. Drafts will launch, and you will be prompted to select an action group for the new action.
