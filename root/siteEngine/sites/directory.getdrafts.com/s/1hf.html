TaskPaper+

Posted by agiletortoise, Last update about 1 year ago

TaskPaper markup highlighting. Includes support for projects, tasks, notes, tags and other common TaskPaper markup. For details, visit the TaskPaper documentation.

This implementation adds support for [[wiki-style]] links, which are not part of the TaskPaper format proper, but useful in Drafts. For details about using this links, see Drafts Cross-Linking docs.
