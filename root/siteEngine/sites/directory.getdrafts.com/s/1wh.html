MultiMarkdown (Full Text)

Posted by agiletortoise, Last update 2 months ago

Markdown syntax with support for MultiMarkdown extensions, like footnotes, definitions lists and Critic Markup.

This is an alternate version of the built-in version of the MultiMarkdown syntax changed to evaluated the full text of the draft with all changes. This can slow performance in some cases, but if you often use longer drafts with code blocks, or similar extended block elements, it avoids the case where changes in longer drafts can lose syntax formatting for some text temporarily.
