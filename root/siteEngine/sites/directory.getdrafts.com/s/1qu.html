Org

Posted by Will, Last update 8 months ago

Provides (partial) syntax highlighting for .org files. Compatible with Logseq’s implementation of org-mode.

Syntax:

  • * Header 1
  • ** Header 2
  • *** Header 3 etc.
  • *bold*
  • /italics/
  • ~code~
  • =verbatim=
  • +strikethrough+
  • ^^highlight^^
  • _underline_
  • [[link][text]]
  • unordered lists with - or +
  • numbered lists
  • - [ ] checkboxes
  • TODO, DONE, LATER, SCHEULED, DEADLINE tasks
  • Properties, eg. #+title: or :title:
  • #+BEGIN … #+END environments etc.
