Action

Process Meeting Notes

Posted by @nahumck, Last update 8 months ago - Unlisted

Process meeting notes to send task-prefixed (- [ ] ) items to Reminders. Save to specified file in iCloud Drive.

Trusted member
Install

Steps

  • script

    // Process Meeting Notes

// Scan for lines with tasks and send them to Reminders
let d = draft.content;
let lines = d.split("\n");
let n = '';
let list = ReminderList.findOrCreate("Inbox")
for (var line of lines) {
	if (line.includes("- [ ]")) {
		// Reminders Action
		var task = line.replace("- [ ]","");
		task = task.trim();
		var reminder = list.createReminder();
		reminder.title = task;
		let success = reminder.update();
		if (success) {
			console.log("Reminders task created: " + task);
		}
		else {
		}
	}
	else {
	// do nothing with the line
	}
}
