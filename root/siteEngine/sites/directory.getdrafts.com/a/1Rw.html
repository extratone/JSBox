Action

Send to Agenda Inbox

Posted by agiletortoise, Last update about 1 year ago

UPDATES

about 1 year ago

Update documentation URL in description.

Create a note in Agenda in a project named “Inbox”, using the first line as that name of the note and remaining text as the note content. A project named “Inbox” must already exist. To target other projects, this example can be editing changing the “project-title” parameter in the Callback URL step.

For more details on options available in Agenda, visit their URL scheme documentation.

Trusted member
Install

Steps

  • callbackUrl

    template
    agenda://x-callback-url/create-note?project-title=Inbox&title=[[title]]&text=[[body]]&on-the-agenda=true
    waitForResponse
    false
    encodeTags
    true
