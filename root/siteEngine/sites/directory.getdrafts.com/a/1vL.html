Action

Hash Tag Transposition (Bullet Line-Based) (copy)

Posted by @sylumer, Last update 3 months ago - Unlisted

Move the first hash tag and anything after it to the start of the line. This is intended to move hash tags at the end of a line to the start of a line.

Trusted member
Install

Steps

  • script

    draft.content = draft.lines.map(strLine => strLine.replace(/^(\- |\* |\+ )(.*?) \#(.*)/, "$1#$3 $2")).join("\n");
draft.update();

Options

  • After Success Default
    Notification Error
    Log Level Error
