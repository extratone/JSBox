Action Group

Markdown

Posted by Bert, Last update about 3 years ago

A markdown action directory. Mostly the same as what comes preinstalled, except the list-like actions will switch between different list types. For instance, if on a line beginning with a dash, the task action will replace the dash with “- [ ] ”, and vice versa. Also works with selections. Numbered lists will be numbered automatically, taking nesting into account.

Experimental (v0.2, untested) – use at your own risk.

Install

Included Actions

  • Markdown Header (#)
  • Outdent
  • Indent
  • Markdown List
  • Toggle numbered lists
  • Tasks
  • List separator
  • Markdown Bold (**)
  • Markdown Emphasis (_)
  • Markdown Quotation (>)
  • Markdown Link
  • Dark-Light Preview
