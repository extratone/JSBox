Action Group

OF Taskpaper

Posted by RosemaryOrchard, Last update about 2 years ago

A series of actions to help you create tasks or projects in Taskpaper format and send them to OmniFocus. It’s designed as a keyboard for easy access. For more information about each action see here: https://www.rosemaryorchard.com/blog/using-drafts-5-taskpaper-with-omnifocus
The cursor is now positioned correctly after using the actions which insert text.
