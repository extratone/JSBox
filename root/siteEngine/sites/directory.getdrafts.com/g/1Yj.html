Action Group

Scene Writer

Posted by Stuart Ottenritter, Last update over 2 years ago

Draft a simple scene with this keyboard and preview setup.

Use the keyboard actions to denote scene headings, characters, and dialogue. Action lines are rendered with the paragraph style; ie no Markdown header. The CSS renders the body of the draft to appear as a screenplay or movie script. Great for writing skits and YouTube videos.

Install

Included Actions

  • Scene Header h1
  • Character h2
  • Dialogue h3
  • Parenthetical h4 + ()
  • Transition h5
  • Preview
  • Scene Preview
  • Teleprompter Preview
