Action Group

Reading Later Actions

Posted by tommertron, Last update over 1 year ago

An Action Group designed to help Drafts be used as a “read later” type service.

Instructions

  1. Install the complementary Workspace
  2. Install the iOS Shortcut to add links you find to the “Read Later” workspace.
  3. Install the iOS Shortcut to add notes to your currently in progress (flagged) draft.
Install

Included Actions

  • Read current item
  • Mark as read
  • To Read
