RochelleB-S:

Stumbled across this while trying to figure out how to make a Drafts action execute a Keyboard Maestro macro. Although I still haven‚Äôt figured that out, I now know I can executive a keyboard maestro action from within a Draft, which hadn‚Äôt occurred to me! Thank you.

You could create an action with a Run AppleScript step and add the Keyboard Maestro code from the macro you want to run. It might looks something like this:

on execute(draft)
	tell application "Keyboard Maestro Engine"
		do script "26BFEDCE-EAC3-4F46-9252-4F6861072243"
	end tell
end execute

Alternatively, you could use the URL action step and the URL trigger for the Keyboard Maestro macro.

e.g. The macro above is triggered by kmtrigger://macro=26BFEDCE-EAC3-4F46-9252-4F6861072243
