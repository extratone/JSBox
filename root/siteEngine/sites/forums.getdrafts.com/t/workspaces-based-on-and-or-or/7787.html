Do you mean that you want a workspace which displays drafts whose tags conform to the expression x AND (y OR z)? As it is, Drafts can‚Äôt encapsulate that logic purely with a workspace tag-filter, though you could programmatically construct a list of matching drafts via script whenever necessary.

  • x AND z NOT y which in Drafts is x,z,!y with the operator set to All.
