Unsupported Syntax in Drafts650×614 38.1 KB

agiletortoise May 21, 2021, 4:00pm #2

You deleted a custom syntax definition. When you try to open a draft which was assigned to that syntax, it cannot be found and applied to the draft, therefore you need to select an alternate syntax from those available in your installation.

I expect, in this case, you have the default syntax for new drafts on iOS set to this syntax you deleted…which is problematic. Change that selection in Editor Preferences (Aa).
