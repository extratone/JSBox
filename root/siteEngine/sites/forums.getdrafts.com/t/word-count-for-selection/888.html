draft8 May 10, 2018, 9:59pm #2

In the interim, you could use a script action along these lines:

http://actions.getdrafts.com/a/1IW

(() => {

	const main = () =>
		alert(
			words(editor.getSelectedText())
			.length
			.toString() + ' words in selection'
		);
	
	// words :: String -> [String]
	const words = s => s.split(/\s+/);

	// MAIN ---
	main();
	
})()
