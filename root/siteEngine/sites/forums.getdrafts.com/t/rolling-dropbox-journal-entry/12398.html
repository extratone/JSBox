The standard Basic action group that comes at install has a rolling Dropbox journal action. It is just the filename that determines how often it roles. The action is monthly, but if you delete the month from the filename it would become yearly.

The original action group is also available in the directory if you have deleted it.

directory.getdrafts.com

Drafts Directory: Basic

Example actions and other extensions for Drafts, the quick-capture notes app.

The action is called “Append to Dropbox Journal”.
