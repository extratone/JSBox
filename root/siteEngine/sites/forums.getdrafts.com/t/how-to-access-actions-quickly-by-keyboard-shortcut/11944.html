sylumer January 16, 2022, 2:03pm #2
  • There is no command palette in Drafts.
  • You can use OPTION + CMD + L to place the focus in the action list search box, trigger the search, press tab and then arrow key to the desired action and press return.
  • You can add a specific keyboard shortcut to each action you wish to trigger.
  • You could create an action that gives you a web pop up and then build a type ahead field to trigger the action.
