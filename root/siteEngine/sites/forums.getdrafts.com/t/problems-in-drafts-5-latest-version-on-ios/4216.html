I’m not aware of any issues with swiping for the action list. The behavior is slightly different in feel from Drafts 4 because it is an interactive swipe, but similar functionality is there. The momentum is a little different, could it just be an adjustment to the new swiping behavior?

The action list disappearing behavior should be the same as in Drafts 4. It remains visible if the action does nothing on success. If the action is configured to archive or trash after success, a new draft is created (or the editor move to next draft in focus mode) and the action list is hidden.
