I wanted to work at remembering and incorporating better vocabulary words into my daily usage. When I encounter a word I want to remember I drop it at the end of a draft (you can use iOS Shortcuts to just appendâ€¦). This action will (1) add an index number to the end of each word, (2) alphabetize the list and (3) randomly choose out three words to be repeated right after the title. (4) It also updates the index so that each time you see a word it is that much less likely to reappear.
The point is to then add these as a list widget. Set the parameter in the widget to [[line|3â€¦5]] to select out those lines.
