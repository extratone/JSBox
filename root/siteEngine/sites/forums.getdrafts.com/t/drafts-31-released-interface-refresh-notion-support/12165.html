Interface Refresh

Lots of general usability and interface and improvements, including updated icons, some button re-organization, and a variety of new features. Highlights:

Notion Integration (Beta)

This version introduces basic integration with the Notion online project management system. Notion‚Äôs API is still in beta but has become stable enough to start offering integration in Drafts. This initial integration provides:

  • Notion Action Step that allows easy configuration of actions which create or append to pages in Notion as either text or todo lists.
  • The Notion script object, which provides access to the entire Notion API for advanced use cases.

For details, example actions, etc., see our Notion Integration Guide.

Maintenance Tasks

New ‚ÄúMaintenance‚Äù view in Settings/Preferences. This view is a home for completely optional periodic maintenance tasks to help restore defaults, clean up data you may no longer care about, or improve performance. Make sure you understand what the tasks do before running one because some are destructive and not reversible. Details on available tasks in the User Guide.
