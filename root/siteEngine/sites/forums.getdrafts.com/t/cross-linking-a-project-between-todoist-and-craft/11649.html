Cross linking a project between todoist and craft

Actions - Share What You've Made
FlohGro November 25, 2021, 8:22pm #1

this action is only useful for Todoist and Craft users. It could maybe be adapted for other Task managers or note apps (totally possible with Drafts itself) but I haven’t looked into this deeper.

I developed an action which is really useful when you often create project documents in Craft and would like to link them to a corresponding project in todoist

It prompts you for a title of the project and then creates the new project in todoist and a new document in Craft (based on a template) which are linked to each other.

The Drafts action can be downloaded here: create crosslinked project in Craft and Todoist | Drafts Directory
