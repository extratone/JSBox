Not exactly what you have described, but this might be an option for you.

directory.getdrafts.com

Drafts Directory: Title Search

Example actions and other extensions for Drafts, the quick-capture notes app.

On running, it will give you a prompt to enter the title text. Select OK (the default button) to proceed, and it will create and load a temporary workspace with the title search string set.

So like I say, not exactly what you requested, but it should give the same overall result of a title search.
