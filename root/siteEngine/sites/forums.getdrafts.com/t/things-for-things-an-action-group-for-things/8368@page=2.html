tommertron January 29, 2022, 3:55pm #24

Here you go!

tell application "Things3"
	get name of projects where name of areas is "Work" and status is open
end tell

I run this in Keyboard Maestro, which gets me a list of comma-delimited projects, which then get split into their own lines, added to a file, then the contents of that file replace whatever is in my Drafts projects list. It‚Äôs very Rube-Goldberg, but it gets the job done.
