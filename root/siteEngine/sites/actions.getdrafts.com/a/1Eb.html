Action

Convert To HTML

Posted by bennettcomfort, Last update almost 4 years ago

This action is duo-scripted and transforms markdown writing into HTML format.

The HTML is formated using the Marked markdown script from drafts 4. The HTML transformed text can be integrated into Evernote, Ulysses, Bear, and any other app that allows for HTML-RTF conversion.

Additionally the html is copied to clipboard
