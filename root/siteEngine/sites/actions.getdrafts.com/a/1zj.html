Action

Workspace Table of Contents

Posted by agiletortoise, Last update 6 days ago

Create a workspace table of content in an HTML preview using only Mustache templates and no script.

Edit the first “Define Template Tag” value to set the name of a valid workspace in your installation.

This action maybe be useful on its own, but is intended mostly as an example of some of the possibilities offered by Mustache templates.

Trusted member
Install

  • defineTemplateTag

    name
    workspaceName
    template
    DEBUG
