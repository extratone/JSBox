Link Helpers

Posted by jochi, Last update almost 2 years ago

These are markdown link helpers — UPDATED for [[wiki style linking]].

  1. Select Drafts to Link UPDATED — Select drafts from customizable Workspace and insert markdown links
  2. Copy Markdown Link — Copy link to selected draft in markdown format
  3. Link to New Draft UPDATED — Create a new draft with the selected text as it’s title, and insert markdown crosslinks in both current and new draft
  4. Find Draft to Link NEW — Search drafts (with selection if exsits) to insert [[wiki style]] link to at cursor.
