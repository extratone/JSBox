Action Group

Find Again

Posted by @mattgemmell, Last update almost 2 years ago

By Matt Gemmell.

Provides a Find dialog (Cmd-Alt-F) with options for case-sensitivity and regular expressions, plus keyboard shortcuts for:

  • Find Next (Cmd-G)
  • Find Previous (Cmd-Shift-G)
  • Enter Selection For Find (Cmd-E)

The Find Next/Previous actions can be invoked repeatedly, without requiring to go via the Find window, making it very easy to jump between occurrences in a draft via the keyboard without interruption. The operations wrap around the end/beginning of the draft too.
