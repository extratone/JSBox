Syntax file for MediaWikis WikiSyntax (ver. 0.2).

Matches most of the common WikiSyntax markup.

There are some limitations:

  • Arbitrary template parameters are too complex to be parsed reliably.
  • Template parameters with nested template tags are not always marked correctly.
  • Support for tables is limited. Table cells will only be marked if they are on separate rows beginning with |
  • Image and category markup is only correctly marked for english and german versions.
