# 功能

- 将网页或 URL Scheme 制作成桌面快捷方式
- 支持从相册选择照片作为图标
- 支持从 JSBox 内置的图标中选取素材并选择颜色制作图标
- 点击桌面图标之后可以直接启动对应的 URL，无须打开 JSBox
